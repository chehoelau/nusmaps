# `nusmaps`
Stay in the loop for NUS event

Website: [nusmaps](nusmaps.com)

Github: [github](https://github.com/chehoelau/nusmaps)

# How to start

Clone `nusmaps`:

```sh
git clone https://github.com/chehoelau/nusmaps.git
```

Install all dependencies:

```sh
cd nusmaps
npm install
```

Create a `.env` in the root folder and provide your secrets from firebase console

Example:

```jsx
REACT_APP_API_KEY=""
REACT_APP_AUTH_DOMAIN=""
REACT_APP_PROJECT_ID=""
REACT_APP_STORAGE_BUCKET=""
REACT_APP_MESSAGING_SENDER_ID=""
REACT_APP_APP_ID=""
```

Run your App:

```sh
npm run start
```
